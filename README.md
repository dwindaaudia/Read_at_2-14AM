# Read at 2:14 AM

A narrative mystery iOS game where a missing best friend's account reads messages every night at 2:14 AM.

## Tech Stack
- SwiftUI — visual transitions, flash, blackout
- UIKit — haptics
- GameplayKit — branching logic & player state
- SpriteKit — glitch animations & audio
- Foundation Models — dynamic message generation for Alex

## Player Paths
Trust · Denial · Avoidance
